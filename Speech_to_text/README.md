# Speech to Text - Portable USB Application

A portable speech-to-text application that runs entirely from a USB drive without installation. Record audio or upload existing audio/video files and convert them to text using OpenAI's Whisper model (via faster-whisper).

### Run
   ```bash
   python src/main.py
   ```

### Build Portable Version

1. Run `build.bat` to create the portable executable
2. Find the output in `dist/SpeechToText/`
3. Copy the folder to your USB drive
