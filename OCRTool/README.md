
### 2. Tesseract OCR Engine


1. Download the Tesseract installer from:
   - Official: https://github.com/UB-Mannheim/tesseract/wiki
   - Direct link: https://digi.bib.uni-mannheim.de/tesseract/tesseract-ocr-w64-setup-5.3.3.20231005.exe

2. Run the installer and note the installation path (default: `C:\Program Files\Tesseract-OCR`)

3. Add Tesseract to your system PATH:
   `C:\Program Files\Tesseract-OCR`

4. Run
   ```bash
   cd OCRTool
   python main.py
   ```
